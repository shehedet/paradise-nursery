# Paradise Nursery 🌿

**Where Green Meets Serenity**

A React-based e-commerce shopping frontend application for a plant nursery, built as the final project for the IBM Full Stack JavaScript Developer course on Coursera.

## Project Overview

Paradise Nursery is a front-end shopping application that allows users to:
- Browse a curated selection of houseplants across multiple categories
- Add plants to a shopping cart
- Manage cart items (adjust quantities, delete items)
- View total cost in real time

## Pages

- **Landing Page** — Hero section with company introduction and a "Get Started" button
- **Product Listing Page** — 18 plants across 3 categories with Add to Cart functionality
- **Shopping Cart Page** — Full cart management with quantity controls, delete, and checkout

## Tech Stack

- React (Vite)
- Redux Toolkit
- React Router DOM
- CSS (custom, no UI library)

## Deployment

Deployed via GitHub Pages using the `gh-pages` package.
